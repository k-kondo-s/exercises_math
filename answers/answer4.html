Answer 4

以下の$f(x)$が最小値をとる際の$x$を、導関数を用いて求めよ。
  1. $f(x) = x^2 - 2x + 1$
    2. $x=1$
  2. $f(x) = 2x^2 - 3x + 5$
    3. $x= \frac{3}{4}$
  3. $f(x) = 3x^2 + 5x -7$
    4. $x=-\frac{5}{6}$
  4. $f(x) = \frac{1}{2}x^2 + \frac{3}{5} + 4$
    5. $x=\frac{3}{5}$
  5. $f(x) = 2(x-1)(x+3)$
    6. $x=-1$
