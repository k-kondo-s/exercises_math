Math Exercises

Exercise 5

以下の条件に対し、勾配降下法の式 $$x_{n+1} = x_{n} - \alpha f'(x_{n})$$ における$x_{1}$、$x_{2}$、$x_{n}$を求めよ。
  1. $f(x) = x^2 - 2x + 1, x_{0} = 1.5, \alpha = 0.1$
  2. $f(x) = 2x^2 - 3x + 5, x_{0} = -1, \alpha = 0.1$
  3. $f(x) = 3x^2 + 5x -7, x_{0} = 0, \alpha = 0.2$
  4. $f(x) = \frac{1}{2}x^2 + \frac{3}{5}x + 4, x_{0} = -9, \alpha = 0.01$
  5. $f(x) = 2(x-1)(x+3), x_{0} = 4, \alpha = 0.1$
